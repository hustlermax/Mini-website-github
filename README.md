# Mini Govil Astro Tarot House

This is the production-ready static website bundle for Mini Govil (Astro Tarot House).

## Included files
- `index.html` - full site and styling in a single file (black/gold premium design)

## Local preview
1. Open folder in VS Code
2. Install and run Live Server extension
3. Open: `http://127.0.0.1:5500/index.html`

## Deploy (Netlify/Vercel/GitHub Pages)
### Netlify
1. `git init` and commit files.
2. Create new site from Git and connect repository.
3. Build command: none. Publish directory: `/`.

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in project root.
3. Choose default settings and it deploys.

### GitHub Pages
1. Commit and push to GitHub.
2. In repo settings > Pages > branch `main` / root.
3. Wait for deployment URL.
